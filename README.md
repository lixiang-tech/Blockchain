# 1.create a new folder
```shell
mkdir hardhat-tutorial

```
# 2.initialize an npm project
```shell
npm init -y
```
# 3.install Hardhat
```shell
npm install --save-dev hardhat
```
# 4.Run Hardhat
```shell
npx hardhat init
```
